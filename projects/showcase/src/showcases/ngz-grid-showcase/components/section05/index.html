Pagination

Pagination can be modified by providing template bindings inside the <ng-container>

Custom pagination template bindings that are available include:

  • [hasPagination] - set to true if you want to display & set custom pagination options
  • [defaultPageLength] - number of items per page displayed by default
  • [pageLengthOptions] - include other options of items per page

All pagination template bindings are optional and support a default fallback options

If you don't want to display pagination explicitly set the [hasPagination] template binding to "false"
