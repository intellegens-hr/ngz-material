<ngx-intellegens-grid />

The ngx-intellegens-grid provides a custom table solution based on Angular Material.

The table comes with a set of predefined optional features which include filtering, sorting, pagination, internal and external error handling and loading.


Working with ngx-intellegens-grid


1. Add ngx-intellegens-grid component

Begin by adding the <ngx-intellegens-grid> component to your template.


2. Provide data to the template

The simplest way to provide data to your table is by passing it a data array.

<ngx-intellegens-grid [dataSource]="myArray"></ngx-intellegens-grid>
