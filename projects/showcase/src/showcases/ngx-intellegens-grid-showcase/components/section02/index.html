[dataSource] attribute

The [dataSource] attribute can accept data in a number of formats:

  • - Raw array of data objects, represented each in its own row
  • - A Promise resolving into an array of data (or failing into an Error)
  • - An Observable resolving into an array of data (or failing into an Error)

When having been passed a Promise as [dataSource], the grid will enter and stay in an internal "loading" state until the promise is resolved or fails with an error. If the Promise fails with an error the grid will enter an error state which it will remain in until the [dataSource] has been updated.

When having been passed an Observable as [dataSource], the grid will enter and stay in an internal "loading" state until the Observable has been resolved with is resolved with data or fails with an error. Every time new data is resolved, the grid will automatically update displayed values. If the Observable fails with an error the grid will enter an error state which it will remain in until the [dataSource] has been updated.
