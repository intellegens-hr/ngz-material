[header] and [footer] attributes

The table also supports custom headers and footers

Headers and footers can be modified by providing template bindings inside the <ng-container>

<ng-container ngxIntellegensGridColumnDef="location" header="Location" footer="Location"><ng-container>

If there is no explicit header and/or footer provided the table will automatically fall back to it's default value, which is shown in the example with the header of the "firstName" column.
