Filtering

Filtering can be added by providing the template binding [hasFiltering]="true" inside the <ng-container>

[hasFiltering] is set to "true" by default

Each row also supports filtering by setting the binding [hasFiltering]="true"

In case the [hasFiltering] is set to "false" the filter field will be visible but disabled

In case all [hasFiltering] bindings inside the column definitions are set to "false" the filtering header will be hidden, same if the [hasFiltering] binding of the table configuration was set to "false"
